# G2i Trivia Game
---

Trivia game using react, redux, and webpack.

---
## How to run the game

1. `npm install` in project home directory
1. `npm run dev`

## How to build the app

1. `npm install` in project home directory
1. `npm run build`
---
#### If NPX is not found

1. `npm install -g npx`
1. `npm run build` in project home directory
